# Issue: ECU前后端下载与授权页面改造

## Goal

按用户要求将前端下载与开通交互简化为搜索+下载/功能授权模式，同时后台新增学习资料图文管理。

## 背景/问题

当前桌面端接线图和资料下载仍是卡片详情模式，授权后台虽已是按功能保存，但页面仍保留车型/ECU 视图语义；开通弹窗也还显示复制按钮和联系备注。用户还要求增加学习资料模块，支持后台发图文文章、前端查看。

## 验收标准 (Acceptance Criteria)

- [x] WHEN 用户在前端搜索接线图型号，系统 SHALL 在大框区域显示搜索结果并在多结果时让用户选择后直接浏览器下载
- [x] WHEN 管理员在后台授权页面操作，系统 SHALL 只按功能授权并支持一键全部授权
- [x] WHEN 用户打开文件下载，系统 SHALL 只保留搜索+下载闭环，不展示复杂卡片框架
- [x] WHEN 用户打开开通弹窗，系统 SHALL 放大字体且不再显示复制收款码链接与联系备注
- [x] WHEN 管理员维护学习资料，系统 SHALL 支持新增/编辑/删除图文文章，前端可查看，且展示风格与文件下载/接线图查询保持一致
